## UTFGrid renderer

This is a renderer for the UTFGrid spec (https://github.com/mapbox/utfgrid-spec), written in managed C++ and C#.

To run this, open the project (written in VS2010) and run the console app 'utfgrid'.

This will output a string of json, like this:
